# Bass Separator

**Bass Separator**는 DSP, ML, DL 기술을 활용해 믹스된 오디오 트랙에서 **베이스(Bass)** 신호를 정밀하게 식별하고 분리하는 프로젝트입니다.

## 프로젝트 목표 (Project Goal)
이미 악기 소스들이 하나로 합쳐져 있는, 발매된 음원 및 녹음 파일 등에서 베이스 기타 소스를 정확하게 분리해내어 별개의 트랙으로 받을 수 있도록 합니다.

1. 분리 알고리즘 실험: 악기 소스 분리에 가장 적합한 알고리즘 및 모델 탐사
2. 모델 파인튜닝 및 개선: 기존 모델에서 베이스 분리의 취약점을 찾아내고 이를 집중적으로 보완
3. 모델 배포 및 서비스 구현

[`로드맵`](./docs/roadmap.md)

## 예상 활용 방안 
1) 믹싱, 마스터링 엔지니어: 합쳐진 트랙에서 원하는 소스를 분리하여 따로 플러그인을 걸거나 음, 박자 조정을 하는 등 음악 제작에서 도움을 얻을 수 있음
2) 아티스트, 세션: 룸 라이브 레코딩에서 원하는 악기 트랙만을 분리하거나, 트랙 분리 및 편집을 용이하게 할 수 있음. 
3) 취미 악기 연주자 및 컨텐츠 제작자: 특정 악기만 빠져있는 Backing Track을 만들거나, 원하는 악기 카피에 활용할 수 있음 
4) 대중 소비자: 이큐 조정처럼 원하는 음원에서 원하는 악기를 분리해 내어 강조하거나 줄이는 등 취향에 맞춘 자체 간이 믹싱 가능
